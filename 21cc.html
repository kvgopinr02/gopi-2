Palindrome Checker C Code

Algorithm

  1. Start the program.
  2. Prompt the user to enter a string.
  3. Calculate the length of the string.
  4. Loop through the first half of the string:
    • If any character does not match its corresponding character from the end, print "not a palindrome" and break the loop.
  5. If the loop completes without breaking, print "palindrome".
  6. End the program.

C Code

#include <stdio.h>
#include <string.h>

int main() {
    char str[45];
    printf("string: ");
    scanf("%s", str);
    int i, n = strlen(str);
    for(i = 0; i < n / 2; i++) {
        if(str[i] != str[n - i - 1]) {
            printf("not a palindrome\n");
            break;
        }
    }
    if(i == n / 2) {
        printf("palindrome\n");
    }
    return 0;
}
