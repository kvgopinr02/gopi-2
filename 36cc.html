Read Last N Characters from File in C

Algorithm

  1. Check if the correct number of arguments is provided.
  2. Open the specified file for reading.
  3. Seek to the end of the file to determine its length.
  4. Seek to the position to read the last `n` characters.
  5. Read and display the last `n` characters from the file.
  6. Close the file.
  7. End the program.
C Code:

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    int ch;
    char *file_name = argv[1];
    int n = atoi(argv[2]);
    FILE *fp = fopen(file_name, "r");
    fseek(fp, 0, SEEK_END);
    long int length = ftell(fp);
    fseek(fp, length - n, SEEK_SET);
    while ((ch = fgetc(fp)) != EOF) {
        putchar(ch);
    }
    fclose(fp);
    return 0;
}
