Number Search in Array C Code

Algorithm

  1. Start the program.
  2. Prompt the user to enter the size of the array.
  3. Declare an array of size n.
  4. Prompt the user to input n numbers into the array.
  5. Prompt the user to enter a number to search.
  6. Loop through the array:
    • If the current element equals the search number, print the index and set a flag to indicate the number was found.
  7. If the number was not found, print "No".
  8. End the program.

C Code

#include <stdio.h>

int main() {
    int r = 0, n, m;
    printf("size: ");
    scanf("%d", &n);
    int arr[n];
    printf("numbers: ");
    for(int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }
    printf("number to search: ");
    scanf("%d", &m);
    for(int i = 0; i < n; i++) {
        if(arr[i] == m) {
            printf("found at index: %d\n", i);
            r = 1;
            break;
        }
    }
    if(!r) {
        printf("No\n");
    }
    return 0;
}
