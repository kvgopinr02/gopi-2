Matrix Addition Program

Algorithm for Reading a Matrix

  1. Start the program.
  2. Prompt the user for the dimensions of the matrix (rows and columns).
  3. Declare a 2D array to hold the matrix elements.
  4. Loop through each element and read the values from the user.
  5. End the program.
C Code for Reading a Matrix:

#include <stdio.h>
void read1(int mat[10][10], int rows, int col) {
    printf("Enter %d elements: ", rows * col);
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < col; j++) {
            scanf("%d", &mat[i][j]);
        }
    }
}

Algorithm for Matrix Addition

  1. Start the program.
  2. Prompt the user for the dimensions of the first and second matrices.
  3. Read both matrices using the read1 function.
  4. Check if the matrices have the same dimensions.
  5. If they do, loop through each element and add corresponding elements from both matrices.
  6. Display the resultant matrix.
  7. If they do not have the same dimensions, display an error message.
  8. End the program.
C Code for Matrix Addition:

#include <stdio.h>

void display(int mat[10][10], int rows, int col) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < col; j++) {
            printf("%d ", mat[i][j]);
        }
        printf("\n");
    }
}

void additionOfTwoMatrices(int mat1[10][10], int mat2[10][10], int rows, int col) {
    printf("The Addition Matrix is:\n");
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < col; j++) {
            printf("%d ", mat1[i][j] + mat2[i][j]);
        }
        printf("\n");
    }
}

int main() {
    int a[10][10], b[10][10], m, n, p, q;
    printf("Enter the size of the first matrix: ");
    scanf("%d%d", &m, &n);
    read1(a, m, n);
    printf("Enter the size of the second matrix: ");
    scanf("%d%d", &p, &q);
    read1(b, p, q);
    printf("The first matrix is:\n");
    display(a, m, n);
    printf("The second matrix is:\n");
    display(b, p, q);
    if (m == p && n == q) {
        additionOfTwoMatrices(a, b, m, n);
    } else {
        printf("Addition is not possible\n");
    }
    return 0;
}
