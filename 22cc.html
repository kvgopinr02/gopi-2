Add Two Numbers C Code

Algorithm

  1. Start the program.
  2. Prompt the user to enter two numbers as strings.
  3. Convert the first string to an integer using atoi.
  4. Convert the second string to an integer using atoi.
  5. Add the two integers.
  6. Display the sum of the two numbers.
  7. End the program.

C Code

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

int main() {
    char num1_str[100], num2_str[100];
    printf("num1: ");
    scanf("%s", num1_str);
    printf("num2: ");
    scanf("%s", num2_str);
    int num1 = atoi(num1_str);
    int num2 = atoi(num2_str);
    int num3 = num1 + num2;
    printf("Sum: %d\n", num3);

    return 0;
}
