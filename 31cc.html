Access and display the members of the structure

Algorithm

  1. Start the program.
  2. Declare a structure Person with fields for name, age, and salary.
  3. Declare a variable person1 of type Person.
  4. Prompt the user to enter the person's name, age, and salary.
  5. Display the entered information.
  6. End the program.
C Code:

#include <stdio.h>

struct Person {
    char name[100];
    int age;
    float salary;
};

int main() {
    struct Person person1;
    printf("Name: ");
    scanf("%s", person1.name);
    printf("Age: ");
    scanf("%d", &person1.age);
    printf("Salary: ");
    scanf("%f", &person1.salary);
    
    printf("Displaying information about the person:\n");
    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);
    printf("Salary: %.2f\n", person1.salary);
    return 0;
}
