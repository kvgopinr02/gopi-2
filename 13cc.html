Array Input and Output

Algorithm

  1. Start the program.
  2. Declare an integer array arr[20] to hold up to 20 elements.
  3. Declare an integer variable num to store the number of elements.
  4. Prompt the user to enter the number of elements and read the input into num.
  5. For i = 0 to num - 1:
    1. Read an integer value from the user and store it in arr[i].
  6. Print "Array: ".
  7. For j = 0 to num - 1:
    1. Print the value of arr[j].
  8. Print a new line.
  9. End the program.
C Code:

#include <stdio.h>
int main() {
    int arr[20];
    int num;
    scanf("%d", &num);
    for(int i = 0; i < num; i++) {
        scanf("%d", &arr[i]);
    }
    printf("Array: ");
    for(int j = 0; j < num; j++) {
        printf("%d ", arr[j]);
    }
    printf("\n");
    return 0;
}
