GCD and LCM Calculator

Algorithm

  1. Start the program.
  2. Prompt the user to input two integer values.
  3. Calculate the GCD of the two integers.
  4. Calculate the LCM using the formula:
    LCM(a, b) = (a * b) / GCD(a, b)
  5. Display the LCM to the user.
  6. End the program.
C Code:

#include <stdio.h>

int gcd(int, int);

void main() {
    int i, j;
    printf("Enter two integer values: ");
    scanf("%d %d", &i, &j);
    int lcm = (i * j) / gcd(i, j);
    printf("The lcm of two numbers %d and %d: %d\n", i, j, lcm);
}

int gcd(int a, int b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}
