File Reading in C

Algorithm

  1. Declare a file pointer and variables for the filename and a line buffer.
  2. Prompt the user for the filename.
  3. Open the file for reading.
  4. Read and display the contents of the file line by line.
  5. Close the file.
  6. End the program.
C Code:

#include <stdio.h>
#include <stdlib.h>

int main() {
    FILE *file;
    char filename[100];
    char line[1000];
    
    printf("filename: ");
    scanf("%s", filename);
    
    file = fopen(filename, "r");
    printf("Contents of the file:\n");
    while(fgets(line, sizeof(line), file)) {
        printf("%s", line);
    }
    
    return 0;
}
