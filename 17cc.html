Matrix Multiplication Program

Algorithm for Reading a Matrix

  1. Start the program.
  2. Prompt the user for the dimensions of the matrix (rows and columns).
  3. Declare a 2D array to hold the matrix elements.
  4. Loop through each element and read the values from the user.
  5. End the program.
C Code for Reading a Matrix:

#include <stdio.h>
void read1(int a[10][10], int m, int n) {
    printf("Enter %d elements: ", m * n);
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            scanf("%d", &a[i][j]);
        }
    }
}

Algorithm for Matrix Multiplication

  1. Start the program.
  2. Prompt the user for the dimensions of the first and second matrices.
  3. Read both matrices using the read1 function.
  4. Check if the number of columns of the first matrix equals the number of rows of the second matrix.
  5. If they are compatible, perform the multiplication of both matrices.
  6. Display the resultant matrix.
  7. If they are not compatible, display an error message.
  8. End the program.
C Code for Matrix Multiplication:

#include <stdio.h>

void display(int b[10][10], int m, int n) {
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            printf("%d ", b[i][j]);
        }
        printf("\n");
    }
}

void multiplicationOfTwoMatrices(int a[10][10], int b[10][10], int m, int n, int q) {
    int c[10][10];
    printf("The resultant matrix is:\n");
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < q; j++) {
            c[i][j] = 0;
            for (int k = 0; k < n; k++) {
                c[i][j] += a[i][k] * b[k][j];
            }
            printf("%d ", c[i][j]);
        }
        printf("\n");
    }
}

int main() {
    int a[10][10], b[10][10], m, n, p, q;
    printf("Enter the size of the first matrix: ");
    scanf("%d%d", &m, &n);
    read1(a, m, n);
    printf("Enter the size of the second matrix: ");
    scanf("%d%d", &p, &q);
    read1(b, p, q);
    printf("The first matrix is:\n");
    display(a, m, n);
    printf("The second matrix is:\n");
    display(b, p, q);
    if (n == p) {
        multiplicationOfTwoMatrices(a, b, m, n, q);
    } else {
        printf("Multiplication is not possible\n");
    }
    return 0;
}
