Array Analysis

Algorithm

  1. Start the program.
  2. Declare an integer variable size to store the size of the array.
  3. Declare integer variables pos, neg, even, odd for counting.
  4. Declare integer variables spos, sneg, seven, sodd for summing.
  5. Prompt the user to enter the size of the array and read the input into size.
  6. Declare an integer array elements[size].
  7. Prompt the user to enter the elements of the array.
  8. For i = 0 to size - 1:
    1. Read each element into elements[i].
    2. If elements[i] > 0:
      1. Increment pos.
      2. Add elements[i] to spos.
    3. Else if elements[i] < 0:
      1. Increment neg.
      2. Add elements[i] to sneg.
    4. If elements[i] % 2 == 0:
      1. Increment even.
      2. Add elements[i] to seven.
    5. Else:
      1. Increment odd.
      2. Add elements[i] to sodd.
  9. Print the count of positive numbers.
  10. Print the count of negative numbers.
  11. Print the count of even numbers.
  12. Print the count of odd numbers.
  13. Print the sum of positive numbers.
  14. Print the sum of negative numbers.
  15. Print the sum of even numbers.
  16. Print the sum of odd numbers.
  17. End the program.
C Code:

#include <stdio.h>
int main() {
    int size;
    int pos = 0, neg = 0, even = 0, odd = 0;
    int spos = 0, sneg = 0, seven = 0, sodd = 0;
    printf("size: ");
    scanf("%d", &size);
    int elements[size];
    printf("Elements: ");
    for (int i = 0; i < size; i++) {
        scanf("%d", &elements[i]);
        if (elements[i] > 0) {
            pos++;
            spos += elements[i];
        }
        else if (elements[i] < 0) {
            neg++;
            sneg += elements[i];
        }
        if (elements[i] % 2 == 0) {
            even++;
            seven += elements[i];
        } else {
            odd++;
            sodd += elements[i];
        }
    }
    printf("Positive numbers: %d\n", pos);
    printf("Negative numbers: %d\n", neg);
    printf("Even numbers: %d\n", even);
    printf("Odd numbers: %d\n", odd);
    printf("Sum of positive numbers: %d\n", spos);
    printf("Sum of negative numbers: %d\n", sneg);
    printf("Sum of even numbers: %d\n", seven);
    printf("Sum of odd numbers: %d\n", sodd);
}
