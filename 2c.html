Factorial
Algorithm Steps:
  1. Input a number (n).
  2. If n <= 1, return 1.
  3. Otherwise, return n multiplied by the factorial of (n-1).
Python Code
def fact(n):
    if (n <= 1):
        return 1
    return n * fact(n - 1)

print("Enter a number: ")
x = int(input())
print(f"Factorial of {x} = {fact(x)}")

C Code
#include

int fact(int n) {
    if(n == 0) {
        return 1;
    }
    return n * fact(n - 1);
}

int main() {
    int n;
    printf("Enter a number: ");
    scanf("%d", &n);
    printf("Factorial of %d = %d\n", n, fact(n));
}
