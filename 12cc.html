Pascal's Triangle Generator

Algorithm

  1. Start the program.
  2. Prompt the user to input the number of rows for Pascal's Triangle.
  3. For each row:
    1. Print leading spaces to center the triangle.
    2. Initialize variable k to 1.
    3. For each column in the row:
      1. If j == 0 or i == 0, set k = 1.
      2. Otherwise, calculate k = (k * (i - j + 1)) / j.
      3. Print the value of k.
  4. Print a new line after each row.
  5. End the program.
C Code:

#include <stdio.h>
int main() {
    int rows, i, j;
    printf("Number of rows: ");
    scanf("%d", &rows);
    int k = 1;
    for (int i = 0; i < rows; i++) {
        for (int s = 1; s <= rows - i; s++) {
            printf(" ");
        }
        for (j = 0; j <= i; j++) {
            if (j == 0 || i == 0)
                k = 1;
            else 
                k = (k * (i - j + 1)) / j;
            printf("%d ", k);
        }
        printf("\n");
    }
    return 0;
}
